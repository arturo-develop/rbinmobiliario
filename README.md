# Ramos Bitar Inmobiliaria - Landing Page

Landing Page profesional para servicios de rescate jurídico inmobiliario.

## Tecnologías
- React
- TypeScript
- Vite
- Lucide React (Iconos)

## Configuración
Para que el formulario funcione, asegúrate de configurar la variable `VITE_APP_WEBHOOK_URL` en tu entorno de despliegue con la URL de LeadConnector.
